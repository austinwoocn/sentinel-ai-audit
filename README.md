## 项目简介

Sentinel是一款企业级AI Agent合规审计系统，为使用大语言模型（LLM）的企业提供全方位的风险控制、合规审计和实时监测能力。

历时6周，完成了从策略拦截到联邦化审计的完整技术栈。

## 核心能力

### 🔥 合规智能（Compliance Intelligence）
- **DSL规则引擎**: 自定义审计规则语法，<10ms扫描延迟
- **序列模式匹配**: 跨会话操作关联分析
- **实时风险评分**: 多维权重算法，Risk>90自动熔断
- **动态基线**: 基于角色的自适应评分，误报率<0.1%
- **影子策略**: 灰度测试新模式，不影响生产业务
- **TSA时间戳签名**: 数字时间戳+区块链/WORM存证，防抵赖

### 🌐  联邦化审计（Federation）
- **联邦广播**: 跨组织事件安全传播
- **零知识证明**: 验证不泄露隐私
- **跨组织溯源**: 完整操作链路追踪
- **拜占庭容错**: 分布式共识机制

### 🎥 自愈与恢复（Replay & Remediation）
- **操作回放**: 精确重现Agent决策过程
- **错误自愈**: 自动检测并修复异常

### 🔍 多模态审计（Audit）
- 文本、图像、音频全支持
- 跨模态关联分析

### 🚫 策略拦截（Policy）
- 实时拦截违规操作
- 细粒度权限控制
